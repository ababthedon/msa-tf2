================================================================================
  WHAT SCIKIT-LEARN PROVIDES
================================================================================

scikit-learn (sklearn) is used for:
- Classification metrics: accuracy_score, f1_score, precision, recall
- Regression metrics: mean_absolute_error, mean_squared_error, r2_score
- Confusion matrices
- Model evaluation utilities

Size: ~30 MB
Installation time: ~30 seconds

================================================================================
  TROUBLESHOOTING
================================================================================

If "pip install scikit-learn" fails:

Try conda:
  $ conda install -c conda-forge scikit-learn

Or upgrade pip first:
  $ pip install --upgrade pip
  $ pip install scikit-learn
